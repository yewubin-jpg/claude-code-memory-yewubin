## 🚀 核心功能 (Core Features)

### 0. 开放兼容：与其他技能的轻量关联
本技能是一个**独立的、大众化的记忆管理系统**，安装后即可独立使用，无需依赖任何其他技能。同时，它具备智能兼容能力——如果你同时安装了其他技能（如 `productivity-skill`），本技能会自动检测并兼容它们写入 `MEMORY.md` 的数据：
- **梦想与目标**：自动识别并设为最高优先级长期记忆，AutoDream 整理时永不归档、永不剪枝
- **项目数据**：自动为每个活跃项目维护独立的主题文件，并按八大领域归类项目文件夹
- **每日笔记**：AutoDream 自动扫描并提取关键事实，过期内容自动归档

> 本技能只负责"记住"和"整理"，不负责"执行"和"排序"。数据通过共享的 `MEMORY.md` 自然流通，无需额外配置。

### 1. 三层记忆架构 (受 Claude 启发)
像人类大脑一样，分层存储：
- **工作记忆**：当前对话的上下文
- **短期记忆**：近期主题文件 (Topic Files)
- **长期记忆**：全局索引 (MEMORY.md) 与归档

### 2. 自动做梦 (AutoDream)
**受 Claude Code 源码的双门控触发机制启发：**
- **触发条件**：距上次整理 ≥ 24 小时 **且** 距上次整理 ≥ 5 次工作会话
- **智能整理**：自动执行定向、收集、整合、剪枝四个阶段，消除矛盾，把短期记忆转化为长期知识。
- **用户控制**：你也可以随时输入 `/dream` 手动触发。

### 3. 易效能文件系统
- 自动用 **"大主题+日期+子主题"** 规范命名所有笔记和文件。
- 笔记是临时的工作输出，记忆是长期的知识沉淀，两者边界清晰。

## 🛠️ 安装与使用 (Installation & Usage)

### 安装命令
在 OpenClaw 聊天框或终端输入：
```bash
clawhub install claude-code-memory-yewubin
```

### 安全安装协议 (Safe Onboarding Protocol)
安装后，龙虾将严格执行以下安全协议：
1. **静默扫描与备份**：扫描您现有的记忆文件并自动备份，**绝不清零您的旧记忆**。
2. **展示报告与请求确认**：向您展示安装评估报告和功能清单。
3. **执行写入**：**只有在您明确回复"同意"后**，才会将新的记忆机制注入到您的核心配置文件中。不同意则绝对不动任何文件。

### 常用命令
- `/dream`：手动触发记忆整理（做梦）
- `/backup`：手动触发工作空间备份
- `/memory`：查看当前的记忆索引状态
