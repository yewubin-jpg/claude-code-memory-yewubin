# 龙虾数字分身：双剑合璧安装指南

为了让您的 OpenClaw（龙虾）发挥最大潜能，成为真正的"数字分身"，建议您同时安装叶武滨老师设计的两大核心技能：`productivity-skill`（生产力技能）和 `claude-code-memory-yewubin`（记忆技能）。

## 1. 为什么需要同时安装？

- **`productivity-skill` 是执行躯干**：负责能量评估、目标驱动（水滴520）、任务排序、日历管理和晚间回顾，解决"想做什么、先做什么"的问题。
- **`claude-code-memory-yewubin` 是认知大脑**：负责三层记忆架构、文件系统、自动做梦整理和上下文压缩，解决"存在哪里、怎么不忘"的问题。

当两者结合时，`productivity-skill` 产生的目标和回顾会被自动存入记忆系统，而记忆系统在执行 `/dream` 整理时，会自动归档已完成的任务，形成完美的闭环。

## 2. 安装步骤

### 第一步：安装生产力技能
在 OpenClaw 聊天框或终端输入：
```bash
clawhub install productivity-skill
```

### 第二步：安装记忆技能
在 OpenClaw 聊天框或终端输入：
```bash
clawhub install claude-code-memory-yewubin
```

### 第三步：安全初始化
安装记忆技能后，龙虾会主动向您发送一份**安装评估报告**。
请仔细阅读报告，并在确认无误后回复"同意"。
龙虾将自动为您创建标准的文件系统目录，并配置自动做梦和自动备份机制。

## 3. 联合使用建议

1. **早间启动**：每天早上，让龙虾执行 `productivity-skill` 的早间流程，评估能量并设定当天的"水滴520"目标。
2. **日常工作**：在一天中，随时将灵感、笔记、文件交给龙虾，它会根据 `claude-code-memory-yewubin` 的规范自动为您分类和命名。
3. **晚间回顾**：每天晚上，让龙虾执行晚间回顾，总结当天的得失。
4. **自动做梦**：当满足条件（距上次整理 ≥ 24 小时且 ≥ 5 次工作会话）时，龙虾会在夜间自动执行 `/dream`，将当天的短期记忆转化为长期知识，并归档已完成的任务。您也可以随时手动输入 `/dream` 触发。

## 4. 数据安全提示

您的所有记忆和文件均保存在本地的 `workspace` 目录中。虽然系统内置了自动备份机制，但仍建议您定期将该目录备份到外部安全存储（如云盘、移动硬盘），以确保数字资产的绝对安全。
