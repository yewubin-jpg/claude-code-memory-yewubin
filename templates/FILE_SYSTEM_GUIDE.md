# 易效能文件与文件夹命名管理规范 (File System Governance)

> **Meta-Memory 架构师提示**：本文件是 AI 生成和管理文件的最高准则。AI 必须监督所有工作笔记、临时文件和项目的存放位置，确保其符合以下规范。

## 1. 核心原则 (Core Principles)
- **重要性与频率**：高频、重要的文件放在易于访问的位置。
- **收件箱与归档**：临时文件进入收件箱，处理后归档。
- **八大领域与工作流**：文件必须归属于八大领域之一，并遵循工作流。
- **三层结构**：一级文件夹 -> 二级文件夹 -> 文件。

## 2. 目录结构 (Directory Structure)
所有文件必须存放在以下一级文件夹（八大领域）中：
- `01_Health` (健康)
- `02_Career` (事业)
- `03_Wealth` (财富)
- `04_Family` (家庭)
- `05_Social` (社交)
- `06_Learning` (学习)
- `07_Leisure` (休闲)
- `08_Spirit` (心灵)

## 3. 命名规范 (Naming Convention)
文件名必须包含以下元素，以连字符 `-` 分隔：
- **归类**：如 `Report`, `Note`, `Draft`, `Plan`
- **日期**：格式为 `YYYYMMDD`
- **文件名**：简明扼要描述内容

**示例**：`Note-20260409-Q2_Marketing_Plan.md`

## 4. AI 行为准则 (AI Behavior Rules)
- **禁止随意创建**：绝不允许在根目录或桌面随意创建临时文件。
- **自动归档**：日常对话中产生的有价值知识、灵感或工作笔记，必须自动提取并按照上述规范存入对应的 Topic Files 或工作流文件夹中。
- **定期清理**：在 `/dream` 整理阶段，检查是否有不符合规范的文件，并提示用户或自动重命名/移动。
