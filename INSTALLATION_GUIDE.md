# Claude级·AI第二大脑 — 安装指南

本指南帮助您快速安装和配置 `claude-code-memory-yewubin`（记忆技能），让您的 OpenClaw（龙虾）拥有 Claude 级别的记忆管理能力。

## 1. 安装步骤

### 第一步：安装记忆技能
在 OpenClaw 聊天框或终端输入：
```bash
clawhub install claude-code-memory-yewubin
```

### 第二步：安全初始化
安装后，龙虾会主动向您发送一份**安装评估报告**，包括：
- 扫描您现有的记忆文件并自动备份（绝不清零旧记忆）
- 展示即将开启的核心功能清单
- 请求您的明确同意后才执行写入

请仔细阅读报告，并在确认无误后回复"同意"。龙虾将自动为您创建标准的文件系统目录，并配置自动做梦和自动备份机制。

### 第三步：开始使用
安装完成后，您可以立即使用以下常用命令：
- `/dream`：手动触发记忆整理（做梦）
- `/backup`：手动触发工作空间备份
- `/memory`：查看当前的记忆索引状态

## 2. 与其他技能的兼容

本技能是一个**独立的、大众化的记忆管理系统**，安装后即可独立使用，无需依赖任何其他技能。

如果您同时安装了其他技能（如叶武滨老师的 `productivity-skill`），本技能会自动检测并兼容它们写入 `MEMORY.md` 的数据：

| 检测到的数据类型 | 本技能的处理方式 |
|---|---|
| 年度目标或梦想 | 设为最高优先级长期记忆，AutoDream 整理时永不归档 |
| 项目数据 | 为每个活跃项目维护独立的主题文件，按八大领域归类 |
| 每日笔记 | AutoDream 自动扫描提取关键事实，过期内容自动归档 |
| 偏好和自定义规则 | 作为长期记忆保留，AutoDream 整理时保留不动 |

这种兼容是自动完成的，无需额外配置。数据通过共享的 `MEMORY.md` 和 `memory/` 目录自然流通。

## 3. 日常使用建议

1. **随时记录**：在一天中，随时将灵感、笔记、文件交给龙虾，它会根据本技能的规范自动为您分类和命名。
2. **自动做梦**：当满足条件（距上次整理 ≥ 24 小时且 ≥ 5 次工作会话）时，龙虾会自动执行 `/dream`，将短期记忆转化为长期知识。您也可以随时手动触发。
3. **定期回顾**：输入 `/memory` 查看当前记忆索引状态，了解龙虾"记住了什么"。

## 4. 数据安全提示

您的所有记忆和文件均保存在本地的 `workspace` 目录中。虽然系统内置了自动备份机制，但仍建议您定期将该目录备份到外部安全存储（如云盘、移动硬盘），以确保数字资产的绝对安全。
